# Pinterest-Downloader
**這是一個使用簡單、操作便捷的[Pinterest](https://www.pinterest.co.kr) 圖片下載器，讓你輕鬆下載和保存Pinterest上的圖片。**
### 需求條件:
   ```py
   $ python3 -m pip install --upgrade -r requirements.txt   
   ```
### 使用方式:
- 在終端機執行以下程式碼
  ```py
  python main.py
  ```
